   - Text frames: TPE1 (artist), TALB (album), TIT2 (title), TRCK (track number as string)
   - Each text frame: 4-byte frame ID + 4-byte size + 2-byte flags (0x00 0x00) + 1-byte encoding (0x00 = ISO-8859-1) + text bytes
   - After the tag, append a few bytes of MPEG sync (0xFF 0xFB 0x90 0x00 + 16 zero bytes) so dhowden/tag recognizes it as MP3
   - Use `os.MkdirAll` for parent dirs and `os.WriteFile` to create

   - "skips file outside media root": create file outside MediaRoot, call ScanFile, verify no DB rows created (ScanFile returns nil, not error)

Use real SQLite via openTestDB. Test files must be under Config.MediaRoot to pass pathguard. Create a thumbDir via t.TempDir() for art extraction.
