Off (default): playback starts with subtitles off. On: a text subtitle track is enabled automatically when the file has one — picking the subtitle language below when set. Image-based (burn-in) subtitles are never auto-enabled, and switching subtitles off in the player always wins.
With subtitles on by default: only a text track in this language is auto-enabled (no match — subtitles stay off). Blank: any text track qualifies. A code like en, de, ja — three-letter forms like eng work too.
Size, background, and position apply to text subtitles rendered by the player. Image-based (burn-in) subtitles are part of the video picture and can't be restyled.
